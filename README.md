# Easy Shutdown
  An easy tool to shutdown, restart and more..
  Written using Lazarus in ObjectPascal..
## Supported Actions
#####  All actions are done without being root!
  * Shutdown
  * Restart
  * Suspend
  * Hibernate
  * Logout
  * LockScreen

## Other Features
  * Delaying Supported
